## 搬瓦工 KiwiVM 面板能干什么

搬瓦工自研了一套叫 KiwiVM 的控制台，基本功能都有，主要是：

1. 开机 / 关机 / 重启 VPS
2. 一键重装系统（支持 20+ 个 Linux 发行版，Debian、Ubuntu、Rocky 都有）
3. 机房迁移——买了一个地点，可以后台自助切换到另一个机房，数据不丢
4. 快照备份
5. 查看流量使用情况
6. 设置 rDNS（PTR 记录）

机房迁移这个功能说句题外话，挺关键的——很多人买 VPS 是因为不确定哪个机房线路对自己最好。搬瓦工允许你买了之后再慢慢测，省去了"万一选错了"的焦虑。
